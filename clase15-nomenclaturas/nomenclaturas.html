Nombres de Variables y selectores

Tipos de nombres:

  • UPPERCASE: ".MICLASE"
  • lowercase: ".miclase"
  • PascalCase: ".MiClase"
  • camelCase: ".miClase"
  • kebab-case: ".kebab-case" (separacion con guiones medios)
  • snake_case: ".snake_case" (separacion con _)

Especificidad

la especificidad (specificity) es la forma en que los navegadores deciden que regla Css se aplica a un elemento. mientras mas especifica sea la regla, mas peso tendra pero tambien mas dificil de sobreescribir y mantener el codigo limpio.

Los valores de especificidad se calculan sumando 3 columnas (0, 0, 0)

  • La primera representa los ids; por cada ID sumamos un valor al contenedor, eje: (3, 0, 0)
  • La segunda cuenta las clases, sus atributos y pseudo clases (:hover, :root, required )

Metodologia BEM

Esta metodologia se centra en bloques -elementos -modificadores. Los nombres de las clases son en kebab-case separando__los elementos y con --los modificadores. ejemplo: bloque_elemento-modificador

Bloques: cajasque por si solas tienen sentido: header,container, checkbox, boton, footer, menu

Elemento: partes de un bloque que no tienen sentido por si solas. "menu item", list item, checkbox caption, header title

Modificadores: variaciones de la apariencia o comportamientode un bloque o elemento: disable, highlited, checked, size big, color red
