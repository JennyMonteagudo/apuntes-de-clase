Header

Cabecera de una seccion o página. Donde estará el logo, título principal, menú de navegación

Nav

Elementos que nos permiten movernos por el sitio. Identificar claramente las secciones que poseen enlaces de navegacion, y que facilita a los motores de busqueda y usuarios movernos entre paginas. NO todos los links tiene que estar dentro de un nav.

Main

crea un contenedor para el contenido principal de la página

Section

Es donde se agrupan contenidos relacionados entre sí. Normalmente incluye un título y una web se compone por secciones, cada una con su sentido y propósito

Article

Define un contenido independiente. El mismo puede estar compartido y utilizado en otro sitios o secciones.
Ejemplos de Uso:
  • un post de un blog
  • Un articulo de una revista
  • Comentario de un usuario
  • tarjeta de un producto

Aside

Es contenido relacionado a nuestro contenido principal indirectamente. Se usa generalmente para publicidad, links a otros productos similares o cosas así.
  • Publicidad
  • Biografía de un autor
  • Enlaces relacionados
  • Información de perfil

Footer

Pie de página, sitio o sección. Puede haber múltiples por página, siempre abajo.
  • Contacto
  • Términos y condiciones
  • SiteMap
