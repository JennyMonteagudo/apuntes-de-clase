Media Queries

Que son las Media queries

Declaraci[n en Css que nos permite definir como se va a visualizar nuestra web
si el ancho es de x se conporta de tal manera

Sintaxis de MQ

Una MQ se compone de multiples partes

  • Keywords: @mediaes una palabra clave reservada
  • MediaType: es un tipo de medida. Ej: screen, print, all
  • Operador/es: ej: and, not, only
  • Breakpoints: La condicion a cumplirse. Ej> max-width, min-widht
  • Reglas CSS: bloques de reglas

Tipos de Media (Screen/Print)

-Print: Destinado a material de vista de impresion
-Screen: pantallas a color (es el usado por defecto)
-All: Todos los dispositivos.
-Pueden encontrar otros que ya no se usan: tv, speech, braile

Tips para usar media queries

  1. Los MQ sobreescriben las reglas de Css anteriores, PERO no poseen mayor epecificidad que las otras reglas, por eso debemos agrgarlas al final del documento
  2. Deben utilizar los mismos selectores que quieran reemplazar
  3. Pensar bien los breakpoint
  4. Podemos crear una @media para cada selector o uno solo para agrupar selectores

Mobile vs tablet Vs Desktop

-Mobile (55%) 320px-480px
-tablets (2%) 768 - 1024
-Desktop (43%) 1024-1920 o mas
